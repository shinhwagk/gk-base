#搭建docker(1.3.2/CentOS_7) 私有仓库(非安全)
####1. 下载docker提供的私有仓库images
```shell
# docker pull registry
# docker run -p 5000:5000 registry
```
####2. 重启docker使用以下命令来开启docker
```shell
# docker -d --insecure-registry ip:5000 
```
注:ip可以使用域名代替

####3. 制作images
```shell
docker commit a5b940b045c1 192.168.1.183:5000/anenjoy/dns:anenjoy
```
- 192.168.1.183:5000： image所在服务器地，如果是官方的hub部分忽略 
- anenjoy：namespace，被称为命名空间，或者说成是你镜像的一个分类
- dns：这个是镜像的具体名字
- latest：这是此image版本号

以上一系列加在一起组成REPOSITORY名字。在pull和push明确使用dockerhub或私有仓库
####4. push images 到私有仓库
```shell
# docker push ip:5000/centos
```
注:docker commit {images}时 new images的REPOSITORY命名必须是 {ip|域名}:5000/centos 才能push
